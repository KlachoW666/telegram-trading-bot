IMB (Imbalance): Зона несбалансированного объёма (gap в цене). Пример: 1D IMB для BTC (92484–94559) — лонг-зона; недельный IMB — шорт на выходе вверх с слабостью.
FVG (Fair Value Gap): Gap в цене для отката. Пример: 1H FVG на US100 (25595$) — ожидание теста снизу как девиации и разворота.
STB-зона (Strong to Break): Зона прорыва с имбалансом. Пример: Лонг от STB в имбалансе для MINA после теста.
Манипуляции с ликвидностью: Свип лоев/хаев (сбор стопов). Пример: Непротестированная манипуляция внутри для MINA; свип лоя с восходящей подструктурой для MOVR.
ОФ (Order Flow): Поток ордеров. Пример: Лонговый ОФ для APT; нисходящий 1H ОФ для BTC — сигнал шорта.
Пулы ликвидности: Крупные скопления ордеров. Пример: Пул сверху для APT (лонг-цель); неперекрытый пул на 97163 для BTC (шорт-зона).
SSL/PDL (Session Sweep Liquidity / Previous Day Low): Реакция от SSL — локальный разворот; доставка к PDL — шорт-сигнал (для EURUSD).
Другие: BOS (Break of Structure) — прорыв структуры; CHOCH (Change of Character) — смена характера тренда.

Паттерны подтверждаются на LTF: длинные тени = нерешительность; полные тела = сила.

Сигналы на вход:
Признаки силы: Остановка падения + подтверждение на LTF (рост объёма, дивергенция). Пример: Лонг от 1D IMB для BTC при тесте снизу.
Признаки слабости: Замедление роста, нисходящий ОФ. Пример: Шорт BTC в недельном IMB при выходе вверх без импульса.
Реакция от уровней: Тест FVG/STB снизу = лонг; свип пула = разворот.
Подтверждение: Закрепление выше/ниже уровня (например, выше 828$ для XMR — лонг к 1325$); дивергенция на индикаторах (RSI/ОФ).
Отмена сигнала: Снятие пула (для APT — отмена лонга); пробой зоны без реакции.

Выходы и стопы:
Стоп-лосс: За зоной (ниже IMB для лонга; выше пула для шорта). Пример: Отмена при снятии пула (APT).
Тейк-профит: Пул ликвидности / следующий уровень (XMR: 1325$  ; US100: 27740  $). Trailing stop на LTF при импульсе.
Выход по сигналу: Закрытие на слабости (для шорта — при силе снизу).

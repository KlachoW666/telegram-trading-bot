# Telegram Trading Bot для BingX

Полнофункциональный Telegram-бот на aiogram для автоматической торговли на бирже BingX.

## Возможности

- 📊 Глубокий анализ рынка (свечи, индикаторы, стакан)
- 🤖 Автоматическая торговля с настраиваемыми стратегиями
- 📈 Ручная торговля с удобным интерфейсом
- 💰 Демо-режим для безопасного тестирования
- 📱 Удобное меню с reply-клавиатурой
- 📊 Статистика и отчёты о торговле
- ⚙️ Гибкие настройки риск-менеджмента

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

**Примечание для Python 3.13:**
- В `requirements.txt` указан `pydantic>=2.8.0` для совместимости с Python 3.13
- Если возникают проблемы с установкой, попробуйте:
  ```bash
  pip install --upgrade pip
  pip install pydantic>=2.8.0 pydantic-core>=2.23.0
  pip install -r requirements.txt
  ```

3. Создайте файл `.env` и добавьте (см. `env_example.txt`):
```
BOT_TOKEN=your_telegram_bot_token_here
BINGX_API_KEY=your_bingx_api_key_here
BINGX_SECRET_KEY=your_bingx_secret_key_here
```

4. Запустите бота:
```bash
python main.py
```

## Использование

1. Запустите бота командой `/start`
2. Подключите API BingX в настройках
3. Настройте риск-менеджмент
4. Выберите режим торговли (демо/реальный)
5. Включите авто-торговлю или торгуйте вручную

## Безопасность

- API ключи хранятся в зашифрованном виде
- Поддержка демо-режима для тестирования
- Гибкие настройки риск-менеджмента
